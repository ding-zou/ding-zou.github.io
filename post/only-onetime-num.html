剑指offer—只出现一次的数字

题目描述

一个整型数组里除了两个数字之外，其他的数字都出现了两次。请写程序找出这两个只出现一次的数字。

思路

  1. 哈希：使用hash表保存每个数字出现的次数进行累加，累加到2的时候把该元素从hash表中删除，最后留下的就是只出现一次的数

    时间复杂度：O(n) 空间复杂度：O(n)

  2. 排序：排序后，循环判断前后数是否相等，不相等返回，相等跳过一个继续循环(跳过的一个是与前一个相等)

    时间复杂度：O(n) 空间复杂度：O(1) 还要排序一次，时间复杂度大于O(n)

  3. 异或：两个相等的数异或=0，只出现一次的数在数组中异或!=0

    时间复杂度：O(n) 空间复杂度：O(1)

  • 异或

利用自反性 A XOR B XOR B = A xor 0 = A

测试：

public class B{
    public static void main(String[] args) {
        int i = 10 ^ 324 ^ 3223 ^ 325 ^ 324 ^ 3223 ^ 432 ^ 325 ^ 432;
        System.out.println(i);
    }
}
输出：
10

代码与实现

哈希法

我们循环遍历数组，添加到map中，如果添加过了，再次被添加时则删除该元素，它时重复的，最后使用hash的keySet方法获取hash表中所有的元素

//num1,num2分别为长度为1的数组。传出参数
//将num1[0],num2[0]设置为返回结果
import java.util.Map;
import java.util.HashMap;
public class Solution {
    public void FindNumsAppearOnce(int [] array,int num1[] , int num2[]) {
        Map<Integer,Integer> map = new HashMap<>();
        for(int i=0;i<array.length;i++){
            if(map.get(array[i])==null) map.put(array[i],1);
            else if(map.get(array[i])==1) map.remove(array[i]);
        }
        for(Integer n:map.keySet()){
            if(num1[0]!=0) num2[0] = n;//如果num1添加过，添加num2
            else num1[0] = n;//添加num1
        }
    }
}

排序法

先排序数组，然后遍历，如果前后两个元素不相等则添加到num数组，第一次num1添加，第二次num2添加；如果相等，i+2，继续判断下下个元素

由于可能最后一个元素为只出现一次的，所以最后需要判断最后一个元素和倒数第二个元素是否相等，不相等则把最后一个元素添加到num2

public class Solution {
    public void FindNumsAppearOnce(int [] array,int num1[] , int num2[]) {
        int count = 0;
        Arrays.sort(array);
        int i = 0;
        while(i<array.length-1){
            if(array[i]!=array[i+1]){
                num1[0] = count==0?array[i++]:num1[0];
                num2[0] = count==0?num2[0]:array[i++];
                count++;
            }else{
                i+=2;
            }
        }
        int l = array.length-1;
        if(l>=1&&array[l-1]!=array[l]) num2[0] = array[l];
    }
}

异或法

这种方法是最优的

  • 对数组中全部数进行异或，重复出现的数会被异或为0，最后异或的结果就是两个不相等的只出现一次的数

我们怎么根据这个结果获取这两个数？

这两个数不相等，异或!=0，所以它的二进制一定有一位为1，这个数位的下标保存下来，可以用这个数位把数组分成两部分，每部分包含一个只出现一次的数(两数该数位不相等)，这两部分其余的数都是成对出现的重复的数(重复的数该数位相等)，只需要再分别对这两部分异或，得到的两个数就是最终结果

流程：

  1. 异或数组，拿到两个不重复数的异或结果
  2. 找到其中为1的第一个数位
  3. 循环数组判断每一个元素该数位为1还是为0，用两个数接收累异或的值(不需要用数组保存，可以直接累异或)
  4. 最终，这两个值就是最终结果
public class Solution {
    public void FindNumsAppearOnce(int [] nums,int num1[] , int num2[]) {
        //re全部异或完就是两个不重复的数字的异或结果
        int re = nums[0];
        for(int i=1;i<nums.length;i++){
            re = re ^ nums[i];
        }
        //找到re二进制中为1的数位n
        int n = 0;
        while(re!=0){
            if((re&1)!=1) n++;
            else break;
            re = re>>1;
        }
        //根据数位n为1的把数组分成两个，每个数组包含一个不重复的数，并且其他数都是相同的，此时每个数组异或结果就是一个不重复的数
        for(int i=0;i<nums.length;i++){
            if(((nums[i]>>n)&1)==1){
                num2[0] ^= nums[i];
            }else{
                num1[0] ^= nums[i];
            } 
        }
    }
}

扩展—LeetCode136—只出现一次的数字

题目描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1]
输出: 1
示例 2:

输入: [4,1,2,1,2]
输出: 4

思路

三种方法与剑指offer一致，但是该题只有一个不重复的，异或时可以直接异或，结果就是最终元素

代码

排序法
class Solution {
    public int singleNumber(int[] array) {
        Map<Integer,Integer> map = new HashMap<>();
        for(int i=0;i<array.length;i++){
            if(map.get(array[i])==null) map.put(array[i],1);
            else if(map.get(array[i])==1) map.remove(array[i]);
        }
        for(Integer n:map.keySet()){
            return n;
        }
        return 0;
    }
}
哈希法
class Solution {
    public int singleNumber(int[] array) {
        Arrays.sort(array);
        int i = 0;
        while(i<array.length-1){
            if(array[i]!=array[i+1]){
                return array[i];
            }else{
                i+=2;
            }
        }
        int l = array.length-1;
        if(l>=1&&array[l-1]!=array[l]) return array[l];
        return array[0];
    }
}
异或法
class Solution {
    public int singleNumber(int[] array) {
        int res = array[0];
        for(int i=1;i<array.length;i++){
            res  = res ^ array[i];
        }
        return res;
    }
}

扩展—LeetCode137—只出现一次的数字 II

题目描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现了三次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,3,2]
输出: 3
示例 2:

输入: [0,1,0,1,0,1,99]
输出: 99

思路

三种方法一样的思路

  1. 排序法只需要改变前后元素相等时跳跃的值
  2. hash只需要改变判断删除元素的值为3
  3. 这题不能使用异或法，三个相同的数异或还是它本身，考虑位运算
    1. 如果把所有出现三次的数字的二进制表示的每一位加起来,那么每一位都能被3整除;
    2. 最后对每一位余3，该位的结果就是结果数的对应位的值

代码

哈希法
class Solution {
    public int singleNumber(int[] array) {
        Map<Integer,Integer> map = new HashMap<>();
        for(int i=0;i<array.length;i++){
            int m = array[i];
            if(map.get(m)==null) map.put(m,1);
            else if(map.get(m)==1) map.put(m,map.get(m)+1);
            else if(map.get(m)==2) map.remove(m);
        }
        for(Integer n:map.keySet()){
            return n;
        }
        return 0;
    }
}
排序法
class Solution {
    public int singleNumber(int[] array) {
        Arrays.sort(array);
        int i = 0;
        while(i<array.length-1){
            if(array[i]!=array[i+1]){
                return array[i];
            }else{
                i+=3;
            }
        }
        int l = array.length-1;
        if(l>=1&&array[l-1]!=array[l]) return array[l];
        return array[0];
    }
}
位运算
class Solution {
    public int singleNumber(int[] nums) {
        int[] bits = new int[32];
        //把结果每一位累加到数组中保存，只出现一次的数为1的位不能被3整除
        for(int i=0;i<nums.length;i++){
            int n = nums[i];
            for(int j=31;j>=0;j--){
                if((n&1)!=0) bits[j]+=1;
                n = n>>1;
            }
        }
        //re接收余3后对应位的值
        int re = 0;
        for(int m=0;m<32;m++){
            re = re<<1;
            re += bits[m]%3;
        }
        return re;
    }
}

扩展—LeetCode138—只出现一次的数字III

给定一个整数数组 nums，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。

这题异或方法有一点变换，用两个数接收异或结果，再创建数组返回

class Solution {
    public int[] singleNumber(int[] nums) {
        //re全部异或完就是两个不重复的数字的异或结果
        int re = nums[0];
        for(int i=1;i<nums.length;i++){
            re = re ^ nums[i];
        }
        //找到re二进制中为1的数位n
        int n = 0;
        while(re!=0){
            if((re&1)!=1) n++;
            else break;
            re = re>>1;
        }
        //根据数位n为1的把数组分成两个，每个数组包含一个不重复的数，并且其他数都是相同的，此时每个数组异或结果就是一个不重复的数
        int a1=0,a2=0;
        for(int i=0;i<nums.length;i++){
            if(((nums[i]>>n)&1)==1){
                a1 ^= nums[i];
            }else{
                a2 ^= nums[i];
            } 
        }
        return new int[]{a2,a1};
    }
}
